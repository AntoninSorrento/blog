## 基本概念

> 英语有五种主要的句形,学会并掌握这些句型是学习英语的第一步

```
1. 主语—谓语
2. 主语—系动词—表语
3. 主语—谓语—宾语
4. 主语—谓语—间接宾语—直接宾语
5. 主语—谓语—宾语—宾语补足语
```

### 1. 主语

放在`谓动词`之前的成分叫做主语

### 2. 谓语

谓语 = 动词 = 谓语动词

动词包括：

核心：

- 实意动词: 具有实际意义的动词(走、跑、跳...)
- 系动词：be 动词 (am,is,are,was,were) 、等同于 be 动词的那些词

了解：

- 助动词：帮助动词的词，主要是为了帮助动词实现各种时态及语态 (be,have,has,had,did)
- 情态动词 can may must should would

### 3. 宾语

放在`实意动词`之后的成分

### 4. 表语

放在`系动词`之后的成分

### 5. 宾补

补充说明宾语的成分

技巧：宾语和宾语之后成分加 be 动词能构成一个逻辑完整的句子，则为宾补
